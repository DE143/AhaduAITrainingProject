# Assets Folder

This folder contains images and media files for the Task Manager Pro project.

## Required Images

To complete the README, add the following images to this folder:

### 1. banner.png (1200x400 px)
Main banner image for the top of README
- **Option A**: Run `python create_banner.py` (requires Pillow: `pip install Pillow`)
- **Option B**: Create manually using design tools below

### 2. gui-screenshot.png
Screenshot of the GUI application in action
- Take a screenshot while running `python task_manager_gui.py`
- Recommended size: 800x600 or larger
- Show the app with some sample tasks

### 3. cli-screenshot.png
Screenshot of the CLI application
- Take a screenshot while running `python task_manager.py`
- Show the menu and some tasks
- Use a terminal with good contrast

### 4. gui-demo.gif (Optional)
Animated GIF showing GUI usage
- Record a short demo (5-10 seconds)
- Show adding a task and marking it complete
- Tools: ScreenToGif, LICEcap, or Kap

## Design Tools (Free)

If you want to create custom graphics:

- **Figma** - https://www.figma.com (professional design tool)
- **GIMP** - https://www.gimp.org (free Photoshop alternative)

## Quick Screenshot Guide

### For GUI Screenshot:
1. Run `python task_manager_gui.py`
2. Add 3-4 sample tasks with different priorities
3. Press `Windows + Shift + S` (Windows) or `Cmd + Shift + 4` (Mac)
4. Save as `gui-screenshot.png` in this folder

### For CLI Screenshot:
1. Run `python task_manager.py`
2. Add a few tasks and view them (option 2)
3. Take screenshot of the terminal
4. Save as `cli-screenshot.png` in this folder

## Image Optimization

Before committing, optimize images:
- Use https://tinypng.com for PNG compression
- Keep file sizes under 500KB each
- Maintain good quality and readability
